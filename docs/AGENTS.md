# AGENTS.md

Este projeto é o AppBNCC.

Objetivo:
Desenvolver um aplicativo Android em Kotlin para consulta estruturada da Computação Complemento à BNCC.

Regras:
- Não alterar arquivos dentro da pasta /referencias.
- Usar os projetos em /referencias apenas como base de estilo, arquitetura e nomenclatura.
- Priorizar padrões já usados pelo aluno em projetos anteriores.
- Evitar tecnologias que não aparecem nos projetos de referência, salvo quando necessário.
- Preferir nomenclaturas simples como model, adapter, repository, ui, database e viewmodel.
- Não usar arquitetura excessivamente sofisticada sem justificativa.
- Gerar código compatível com Android Studio.
- Em caso de duvida solicitar confirmação.
- Explicar decisões técnicas em /docs/decisoes-tecnicas.md.

Contexto do AppBNCC:
O app terá dados da Computação Complemento à BNCC, incluindo etapas, séries, eixos, objetos de conhecimento, conceitos e habilidades.
